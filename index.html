Du design au Dév. WEB

Après 4 ans dans le domaine du design, j’ai réalisé les opportunités énormes que présente l'industrie technologique pour améliorer nos vies de façon continue. J'ai réorienté ma carrière pour devenir développeur web.

Cela ne veut pas dire que j’ai quitté le design pour autant. C’est grâce à la façon de concevoir une idée en ayant recours au design thinking qu’ont pu naître des entreprises innovantes telles qu’Airbnb, Uber ou encore Deliveroo. C'est dans les interactions et les dynamiques humaines que le design thinking peut innover et créer de la valeur.

Projet Groupomania

Reseau social d’entreprise

Brief : Créer un réseau social pour permettre aux employés de Groupomania de se connaître dans un cadre plus informel.

L’application a été conçue comme une SPA avec React, entièrement avec des composants fonctionnels, react-hooks, react-context (session utilisateur) et react-router. Le back-end a été construit comme une API REST en utilisant Node + Express pour gérer les requêtes, l’authentification, les opérations CRUD et le téléchargement d’images, tout en suivant les standards OWASP de sécurité.

  • Express
  • react.js
Projet Orinoco

MVP Site Web E-commerce

Brief : Développer le MVP pour un site de E-commerce d’appareils photo vintage d’Orinoco.

Le front-end a été développé comme une MPA en utilisant HTML 5, CSS 3, SaSS et Vanilla JavaScript. La page de la liste de produits ainsi que la page de produit sont générées de façon dynamique en obtenant les données depuis le serveur. Le back-end a été construit avec Node.js, Express.js et MongoDB.

  • Express
  • HTML
  • CSS
Projet Vesper Invest

site web vitrine + SEO

Brief : Créer un site web responsive et son référencement pour la société Vesper Invest à Paris.

Conçu mobile-first et responsive à partir des maquettes fournies par le client, le site web a été développé avec HTML 5, CSS 3, SaSS et JavaScript. L’intégration d’OpenGraph, de Twitter cards ainsi que l'indexation du site font partie de l'ensemble du SEO. Côté back-end, le site est déployé sur le service de l’hébergeur du client, avec son certificat SSL et en utilisant le protocole sécurisé HTTPS.

  • HTML
  • CSS
