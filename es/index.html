Mini Bio...

De diseñador a programador

Después de 4 años en el campo del diseño, me di cuenta de las enormes oportunidades que presenta la industria tecnológica para mejorar continuamente nuestras vidas. Para ello, he reorientado mi carrera para convertirme en programador web.

Sin embargo, eso no significa que dejé el diseño. Es gracias a la forma de concebir ideas utilizando el pensamiento de diseño que han podido surgir empresas innovadoras tales como Airbnb, Uber y Deliveroo. Es en las interacciones humanas donde el pensamiento de diseño innova y crea valor en las soluciones propuestas.

Nuevas competencias

Ya sea una necesidad de transformación digital de servicios o concebir una idea nueva e innovadora, es necesario desarrollar el proyecto de la A a la Z lo más rápido y confiable posible. Para ello, me he especializado en tecnologías claves para que el desarrollo de la solución sea rápida, escalable y eficiente.

El Fullstack MERN (MongoDB, Express.js, React y Node) o SERN permiten crear soluciones comerciales de baja y mediana escala, como redes sociales, comercios electrónicos o plataformas web. También estoy en ls capacidad de concebir el UX / UI del sitio, desde las maquetas hasta la experiencia final.

Groupomania

Red Social para los empleados

Brief: Diseñar y programar una red social que permita a los empleados de Groupomania conocerse en un ambiente más informal y amical.

Concebida como una SPA desarrollada completamente en React con componentes funcionales, react-hooks, react-context (sesión de usuario) y react-router.

El backend se programó como una API REST utilizando Node & Express para administrar las solicitudes, la autenticación con JWT, operaciones CRUD y cargas de imágenes siguiendo el top 10 de los diez estándares de seguridad de OWASP.

  • Express
  • react.js
Orinoco

MVP comercio electronico

Brief: Diseñar y programar el MVP de comercio electrónico para el negocio de venta de cámaras antiguas de Orinoco.

La interfaz se programó utilizando HTML 5, CSS 3, SaSS y Vanilla JavaScript. La página de la lista de productos así como la página de producto son generados dinámicamente por JavaScript basado en la information de la base de datos. El backend se programó con Node.js, Express.js y MongoDB.

  • Express
  • HTML
  • CSS
Vesper Invest

site web y SEO

Brief: Programar el sitio web de Vesper Invest de manera que sea responsivo y con su respectivo SEO

Programado mobile-first y responsivo a partir de la maqueta proporcionada por el cliente. El sitio web se programó utilizando HTML 5, CSS 3, SaSS y JavaScript. La integración de OpenGraph y Twitter cards, así como la indexación del sitio forman parte de la estratégia basica de SEO.

La pagina web fue desplegada en el proveedor de hosting del cliente con su respectivo certificado SSL y siempre utilizando el protocolo HTTPS.

  • HTML
  • CSS

Sitio Web con bootstrap

Brief: Diseñar y programar el sitio web para Cycle Agency dentro de un corto plazo de tiempo.

El sitio web fue programado utilizando el framework de Bootstrap y sus componentes en la version 4.5 para obtener una página responsiva y con una buena experiencia de usuario.
