Groupomania

Social Network

Brief: Develop a social network that allows Groupomania's employees to get to know each other in a more informal environment.

The application was conveived as a SPA with React, entirely developed with functional components, react-hooks, react-context (user session) and react-router.

The backend was built as a REST API using Node & Express to manage requests, JWT authentication, CRUD operations and image uploads while following the OWASP top ten security standards.

  • Express
  • react.js
Orinoco

E-commerce MVP Website

Brief: Develop the MVP E-commerce website for the vintage cameras business of Orinoco.

The frontend was developed using HTML 5, CSS 3, SaSS and Vanilla JavaScript . The Product list page as well as product page are dynamically generated by JavaScript based on the database. The back-end was built with Node.js, Express.js and MongoDB.

  • Express
  • HTML
  • CSS
Vesper Invest

Homepage + SEO

Brief: Develop a responsive website with its SEO for Vesper Invest in Paris.

Coded mobile-first and responsive from the mockups provided by the client, the homepage was developed with HTML 5, CSS 3, SaSS and JavaScript. The integration of OpenGraph and Twitter cards as well as the site's indexing were part of the overall SEO basic strategy. On the backend, the website was deployed on the client's prior host provider with its SSL certificate and always using the HTTPS protocol.

  • HTML
  • CSS
